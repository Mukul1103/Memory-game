# Memory Game Project

### How to Start

- Open `index.html` to start
- Click the card to flip it.
- If two cards match, they get locked.
- If they do not match, they flip back.
- You win after you have matched all the cards.

### Project is made using

- HTML
- CSS
- JS
- Bootstrap

## Dependencies
jQuery3.2.1
SweetAlert2
Google Fonts
