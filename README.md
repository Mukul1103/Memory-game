# Memory Game 

### How to Start

- Open index.html to start
- Click the card to flip it.
- If two cards match, they get locked.
- If they do not match, they will flip back.
- You win once you have matched all the cards.

### Project is made using

- HTML
- CSS
- JavaScript
- Bootstrap

## Dependencies
jQuery3.2.1
SweetAlert2
Google Fonts
